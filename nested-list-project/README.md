# רשימות מקוננות - Nested List App

אפליקציית React + Vite ליצירת רשימות מקוננות עם עד 6 רמות קינון ואקורדיון נשלף.

## התקנה והרצה

```bash
npm install
npm run dev
```

האפליקציה תיפתח בדפדפן בכתובת http://localhost:3000

## תכונות

- ✅ יצירת פריטים חדשים
- ✅ עריכת טקסט inline
- ✅ קינון עד 6 רמות
- ✅ אקורדיון פתיחה/סגירה
- ✅ מחיקת פריטים
- ✅ תמיכה מלאה ב-RTL (עברית)
- ✅ עיצוב מודרני עם צבעים לכל רמה

## שימוש

- לחץ על "הוסף פריט חדש" להוספת פריט ראשי
- לחץ על + להוספת תת-פריט
- לחץ על הטקסט לעריכה
- לחץ על ◀ לפתיחה/סגירה של תת-פריטים
- לחץ על ✕ למחיקה

## Snapshots, Templates & Sessions

- פתח את [פאנל הניהול (Side Panel)](#snapshots-templates--sessions) דרך לחצן הצד הצף כדי לגשת ללשוניות **Snapshots**, **Templates** ו-**Sessions**. שמירה, שחזור ומחיקה נעשים דרך מודלי אישור מובנים, כך שתמיד תראה מה בדיוק נשמר או יימחק לפני הפעולה.
- Snapshots שומרים את מצב העץ והסשן הנוכחי לצילום חד-פעמי; השתמש ב-📸 **Create New Snapshot** לפני ניסוי או שינוי גדול.
- Templates ממירים פריט נבחר לתבנית לשימוש חוזר; אם לא נבחר פריט, תראה Toast שמתריע על כך ותוכל לבחור מחדש לפני השמירה.
- Sessions מאפשרים ליצור סשנים מקבילים עם ערכת נושא, כיוון ותצוגה משלהם. מעבר ביניהם מתבצע בלשונית 💼 **Sessions** ויוצג במודל אישור כאשר אתה מחליף או מוחק סשן.

### דוגמאות שימוש קצרות

- רוצה לגבות רגע לפני שינוי: פתח את פאנל הניהול ▶ שמור Snapshot ▶ בצע את השינויים ▶ חזור עם **Restore** אם צריך.
- בניית מבנה חוזר: בחר פריט ראשי ▶ שמור אותו כ-Template ▶ הוסף אותו לסשנים אחרים מבלי לבנות מחדש.

## Command Palette & Keyboard Shortcuts

- פתח את ה-Command Palette עם **Ctrl/Cmd + K** (או לחצן ⌘ Commands בטולבר) כדי להריץ פקודות כמו יצירת פריט, מעבר בין תצוגות, ייצוא או שמירת Snapshot.
- ניווט מהיר: חפש פקודות לפי שם/תיאור, השתמש בחיצים למעלה/למטה וב-Enter להפעלה, או Esc לסגירה. פלטת הפקודות היא מודל במרכז המסך, כך שקל לזהות אותה בזמן העבודה.
- קיצורי מקלדת מובנים: Ctrl+Z לביטול, Ctrl+Shift+Z לחזרה, Ctrl+E לייצוא ל-JSON, Space לכיווץ/פתיחה של פריט, Tab/Shift+Tab לגלישה ברמות, ו-F2 לעריכה inline.

## Recovery & backups

- שמירה אוטומטית: כל שינוי נרשם מקומית ומגובה אוטומטית (כולל Snapshots/Templates/Sessions). אם תסגור את הדפדפן, הנתונים ייטענו חזרה בפתיחה הבאה.
- ייצוא מהיר: השתמש בקיצור Ctrl+E או בפקודת Export בפלטת הפקודות כדי לשמור קובץ JSON חיצוני כגיבוי נוסף או להעברה בין מכונות.
- תזכורות חזותיות: פעולות שמירה/שחזור/מחיקה מפעילות מודלים או Toasts שמסבירים מה קורה. אם שמירה אוטומטית או ייצוא נכשלים, תראה Toast שיידע אותך לנסות שוב או לבצע ייצוא ידני.

## בניית גרסת Production

```bash
npm run build
```

הקבצים יישמרו בתיקיית `dist/`

## יצירת גרסת EXE ניידת (אופליין)

1. התקן תלויות (כולל `pkg` ו-`express`):

   ```bash
   npm install
   ```

2. הרץ בנייה ואריזה לקובץ EXE עבור Windows (Node 18, 64bit):

   ```bash
   npm run package:win
   ```

3. הקובץ `dist/nested-list-app.exe` שייווצר הוא נייד ואינו דורש התקנה. ניתן להריץ אותו ללא חיבור לרשת; הוא מגיש את הקבצים המקומיים של `dist/` על פורט 4173 (או פורט אחר דרך משתנה הסביבה `PORT`).

להרצה מקומית של גרסת ה-Production בלי לבנות EXE, ניתן להשתמש בשרת הסטטי הכלול:

```bash
npm run build
npm run serve:dist
```
